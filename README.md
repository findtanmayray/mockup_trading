# TradeFlow - Trading Management Application

**TradeFlow** is a modern, premium web application designed for active traders to manage their positions, visualize risk, and journal their performance.

## üöÄ Features

### 1. Position Management
- **Unified Dashboard**: View positions from multiple brokers (Zerodha, Fyers, etc.) in one place.
- **Grouping**: Select multiple positions and group them (e.g., "Nifty Hedging") to track combined P&L.
- **Notes & History**: Add notes to any trade or group. View a timeline of events (Entry, Alerts, Exit).
- **Separation**: Distinct views for "Open" and "Closed" positions.

### 2. Strategy Tracking
- **Tagging**: Assign strategies (e.g., "Trend Following", "Scalping") to your trades.
- **Management**: Add or remove custom strategies via Global Settings.

### 3. Advanced Risk Management
- **Define Risk**: Set Profit Targets, Stop Losses, and Trailing Stop Losses for individual or grouped positions.
- **Visual Feedback**: Real-time P&L color coding and stats.

### 4. Trading Journal
- **Yearly View**: A 12-month calendar heatmap showing daily performance at a glance.
- **Drill Down**: Click on any day to see the specific trades and notes that contributed to the P&L.

    ```bash
    git clone https://github.com/yourusername/tradeflow.git
    cd tradeflow
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

3.  Run the development server:
    ```bash
    npm run dev
    ```

4.  Open [http://localhost:5173](http://localhost:5173) in your browser.
